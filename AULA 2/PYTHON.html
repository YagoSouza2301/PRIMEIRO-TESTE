O QUE SIGNIFICA PYTHON?

O Python é uma linguagem de programação amplamente usada em aplicações da Web, desenvolvimento de software, ciência de dados e machine learning (ML).
Os desenvolvedores usam o Python porque é eficiente e fácil de aprender e pode ser executada em muitas plataformas diferentes.

PYTHON É FÁCIL?

Python é uma das linguagens mais fáceis para se começar. Programas feitos com Python parecem com a língua inglesa.
Por causa de sua simplicidade, a maioria dos cursos de nível iniciante usam o Python para introduzir conceito de programação para seus alunos.
